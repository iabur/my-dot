## General Keymaps

| Action         | Shortcut |
|----------------|----------|
| Move Up        | `<C-k>`  |
| Move Down      | `<C-j>`  |
| Move Left      | `<C-h>`  |
| Move Right     | `<C-l>`  |
| Clear Search Highlight | `<leader>h` |

## Completion and Suggestions

| Action                  | Shortcut   |

| Scroll Documentation Down | `<C-f>`  |
| Trigger Completion      | `<C-Space>`|
| Abort Completion        | `<C-e>`    |
| Confirm Selection       | `<CR>`     |
